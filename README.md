<h1 align="center">🖥️ WINDOWS SERVER & ACTIVE DIRECTORY INFRASTRUKTUR</h1>

---

### 📌 Projektbeschreibung
Dieses Projekt simuliert den professionellen Aufbau einer Windows Server-basierten Unternehmensumgebung innerhalb einer virtuellen Hyper-V Infrastruktur. Das primäre Ziel war die erfolgreiche Implementierung einer zentral verwalteten Domäne mit Active Directory sowie allen geschäftskritischen Netzwerkdiensten.

---

### 🧩 Kerninhalte des Projekts

* **💿 Betriebssystem:** Installation und grundlegende Härtung von Windows Server 2025.
* **🌐 Virtualisierung:** Aufbau einer isolierten Server-Client-Umgebung mittels Hyper-V.
* **🛡️ Active Directory (AD DS):** Konzeption und Bereitstellung der zentralen Verzeichnisdienste.
* **🔄 Ausfallsicherheit:** Einrichtung eines Primary und eines Backup Domain Controllers zur Ausfallsicherung.
* **🔍 Namensauflösung:** Konfiguration von Active Directory-integriertem DNS.
* **🤖 Automatisierung (DHCP):** DHCP-Bereitstellung inklusive Failover-Konfiguration (Hochverfügbarkeit).
* **⚙️ Gruppenrichtlinien (GPO):** Erstellung restriktiver Richtlinien zur zentralen Benutzer- und Systemverwaltung.
* **👥 Identitätsmanagement:** Strukturierte Benutzer-, Gruppen- und Rechteverwaltung (IAM).
* **🔐 AD CS (Zertifikatsdienste):** Aufbau einer internen Zertifizierungsstelle für sichere Verbindungen.
* **💻 Administration:** Verwaltung der gesamten Infrastruktur via GUI und automatisierten PowerShell-Skripten.

---

### 🛠️ Technologiestack & Protokolle

* 🖥️ **Virtualisierung:** `Hyper-V`
* 💿 **Betriebssystem:** `Windows Server 2025`
* 🛡️ **Verzeichnisdienst & IAM:** `Active Directory (AD DS)`
* ⚙️ **Zentrale Steuerung:** `Group Policy Objects (GPO)` • `AD CS (Zertifikatsdienste)`
* 🌐 **Netzwerkdienste:** `DNS` • `DHCP (inkl. Failover)`
* 📜 **Automatisierung:** `PowerShell`

---

### 🧠 Erreichte Lernziele

* **Struktur-Verständnis:** Tiefgehendes Verständnis von komplexen Domänenstrukturen und Vertrauensstellungen in Unternehmen.
* **Sicherheitskonzepte:** Praktische Umsetzung zentraler Benutzer- und Rechteverwaltung nach dem Principle of Least Privilege.
* **Infrastruktur-Design:** Kopplung und Absicherung essenzieller Netzwerkdienste in Microsoft-Umgebungen.
* **Effizienzsteigerung:** Erste Schritte zur Automatisierung wiederkehrender Admin-Aufgaben via PowerShell.
